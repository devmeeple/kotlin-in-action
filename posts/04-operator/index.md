# 4. 연산자

## 문제 1 - 정수와 평균

1. `num1`, `num2`, `num3` 변수를 선언하고 각각 10, 20, 30으로 초기화한다.
2. 세 변수의 합을 계산하고 결과를 `sum` 변수에 할당한다.
3. 평균을 계산하고 결과를 `average` 변수에 할당한다. 소수점 이하는 버린다.
4. `sum`과 `average` 변수의 값을 출력한다.
